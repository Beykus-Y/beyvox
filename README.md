# BeyVox

**BeyVox** — это современный кросс-платформенный голосовой мессенджер с открытым исходным кодом, спроектированный по модели self-hosted серверов и единой федеративной авторизации. BeyVox предлагает пользователям полный контроль над своими данными, высокую производительность и отличное качество связи.

## Основные возможности проекта
- **Self-hosted инстансы**: Поднимайте собственные серверы (сообщества) для полного контроля данных и каналов связи.
- **Голосовые и текстовые каналы**: Общайтесь в чатах в реальном времени с поддержкой WebSocket и обменивайтесь голосом через WebRTC/OPUS.
- **Интеграция VST-плагинов**: Применяйте профессиональные плагины обработки звука (VST2/VST3) к вашему микрофону прямо внутри клиента.
- **Стриминг экрана**: Делитесь экраном с высокой частотой кадров с использованием технологии Simulcast.

## Структура проекта

Для удобства разработки проект разделен на независимые модули:

- [**Клиентское приложение (Tauri + Vue 3)**](client/README.md) — кросс-платформенное десктопное приложение для подключения к серверам и общения.
- [**Сервер сообщества (Rust + Axum)**](server/README.md) — серверный инстанс для хостинга собственного голосового и текстового пространства с базой данных PostgreSQL.

Подробную информацию по установке, настройке и запуску каждого из компонентов вы найдете в соответствующих разделах по ссылкам выше.
